# simple_android_projects
Project Overview

A curated collection of basic Java programs designed to demonstrate fundamental programming concepts.
This project is ideal for beginners who are learning Java and preparing for practical exams or lab submissions.

The programs focus on:

User Input Handling

Conditional Statements

Loops

Methods & Data Passing

Arrays

Basic Simulation of Android Concepts
